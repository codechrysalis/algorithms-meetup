# Array Splitter

## Summary

Given a list of numbers, can you separate them into two lists with equal sums?

Reordering the numbers is allowed.

## Input & Outputs

Input is a list of integers.

Output is a list of two integers.

## Example

Input:

```js
[1, 2, 3, 4, 5, 6, 7, 8]
```

Output:

```js
[
  [1, 3, 6, 8],
  [2, 4, 5, 7]
]
```
