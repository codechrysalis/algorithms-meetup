# nQueens

## Summary

Given an N x N chessboard, what is the max number of queens you can place on the board without the queens conflicting with one another?

## Input

N, which denotes the size of the board.

## Output

A number.

## Example

In a 3 x 3 board, the max number of queens you can place is 2.

## Constraints 
 
3 <= N <= 10^5
